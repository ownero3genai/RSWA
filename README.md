# RSWA — Realtors Social Welfare Association

**Official Website of RSWA | Mumbai to Boisar**

---

## About

This is the official corporate website of **RSWA (Realtors Social Welfare Association)** — the premier association of real estate professionals spanning the Western rail corridor from Mumbai to Boisar, Maharashtra, India.

RSWA unites channel partners, builders, and property consultants across Mumbai, Vasai-Virar, Palghar, and Boisar — providing legal support, RERA guidance, training programs, and a powerful professional network since 2010.

## Technology Stack

- **HTML5** — Semantic, accessible markup
- **CSS3** — Custom design system with CSS variables, animations, skeleton loaders
- **Vanilla JavaScript** — No frameworks; scroll animations, counters, search filters, mobile menu
- **Fonts** — Poppins (headings) + Inter (body) via Google Fonts
- **Icons** — Font Awesome 6.5.0
